🔗 공유용 내보내기

라운드를 입력·수정한 뒤 이 버튼을 누르면, 지금까지의 모든 기록과 사진이 담긴 index.html 파일이 만들어집니다. 이 파일을 GitHub 저장소에 덮어씌워 올리면(Commit) 친구들이 링크에서 최신 결과를 보게 됩니다.

사진이 많으면 파일이 커집니다. 친구들은 이 파일을 보기만 하고, 입력·수정은 이 기기에서만 됩니다.

💾 백업 · 복원

전체 기록을 파일로 저장해 두면, 앱에 문제가 생기거나 기기를 바꿀 때 그대로 되살릴 수 있습니다. 라운드가 끝날 때마다 한 번씩 백업해 두시길 권합니다.

복원하면 현재 앱의 기록이 백업 파일 내용으로 교체됩니다. 백업 파일은 안전한 곳(구글 드라이브 등)에 보관하세요.
